## ğŸ® How to Play the Game

1. **Start the Server**
   - Run `Server_Tic-Tac-Toe.exe`.
   - Make sure the server status is **Running**.

2. **Open the Client Application**
   - Launch the Tic-Tac-Toe client on two different machines  
     **or** on the same machine using two separate windows.

3. **Connect to the Server**
   - Enter the **Server IP Address**.
   - Enter the **Port Number**.
   - Click **Connect**.

4. **Login or Register**
   - Each player must log in or create a new account.
   - Wait until both players are connected.

5. **Start a New Game**
   - One player creates a game room.
   - The second player joins the room.

6. **Play Tic-Tac-Toe**
   - Players take turns placing **X** or **O**.
   - The server validates each move.
   - The game ends when:
     - A player wins, or
     - The board is full (draw).

7. **Game Result**
   - The result (Win / Lose / Draw) is displayed to both players.
   - Players can start a new game or exit.
